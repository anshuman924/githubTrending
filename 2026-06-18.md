* [Panniantong / Agent-Reach](https://github.com/Panniantong/Agent-Reach):Give your AI agent eyes to see the entire internet. Read & search Twitter, Reddit, YouTube, GitHub, Bilibili, XiaoHongShu — one CLI, zero API fees.
  - Summary: Agent Reach is a capability layer that installs, configures, and health-checks internet access tools for command-line AI agents. Instead of wrapping content retrieval itself, it selects ordered backends per channel and lets the agent call upstream tools directly, such as Jina Reader for web pages, yt-dlp for YouTube subtitles, gh CLI for GitHub, feedparser for RSS, Exa via MCP for search, and login-backed tools for Twitter/X, Reddit, Bilibili, Xiaohongshu, LinkedIn, V2EX, Xueqiu, and podcasts. Its `agent-reach doctor` command probes real backend availability and reports fixes, while install modes include automatic setup, safe mode, dry run, and uninstall. It is intended for users of coding or shell-capable agents such as Claude Code, OpenClaw, Cursor, Windsurf, or Codex who want their agent to read, search, and summarize online content without manually assembling platform-specific tooling.
